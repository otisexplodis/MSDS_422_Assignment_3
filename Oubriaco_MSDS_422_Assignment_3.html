Stacking Variables¶

This is the first time I've tried stacking variables. I had a few submissions of my work which I'll discuss more in the paper, but it seemed like more stacking the better. The submission I had where I didn't use Age_Class_Sex performed much worse than my other submissions.

The thought behind this stacking was that if we know certain ages, classes, and sex's had a higher likeliehood of surviving, if we stack them together we can get good binary data on specific combinations of these things.

def fare_group(fare):
    a= ''
    if fare <= 4:
        a = 'Very_low'
    elif fare <= 10:
        a = 'low'
    elif fare <= 20:
        a = 'mid'
    elif fare <= 45:
        a = 'high'
    else:
        a = "very_high"
    return a

train['fare_group'] = train['Fare'].map(fare_group)

    corr=df_corr.corr()
